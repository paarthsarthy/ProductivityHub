SETTINGS

▸ STEP 1 — GOOGLE SHEET SETUP
  1. Create a new Google Sheet
  2. Row 1 headers exactly: title channel youtube_id duration category week order tip completed_at
  3. Leave completed_at column empty — app fills it automatically
  4. File → Share → Publish to web → CSV
  5. Copy the URL and paste below
▸ STEP 2 — APPS SCRIPT SETUP (FOR AUTO TIMESTAMPS)
  1. In your sheet: Extensions → Apps Script
  2. Delete existing code, paste the script Claude gives you
  3. Deploy → New deployment → Web App
  4. Set "Who has access" to Anyone
  5. Copy the Web App URL and paste below

Paste the "Publish to web" CSV link from your Google Sheet.

Optional but recommended. Enables auto-timestamp when you mark a video done.
